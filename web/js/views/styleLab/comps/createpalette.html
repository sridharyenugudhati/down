Palette

This page shows the color palette. These variables are not used directly in the component files, rather they are referred to by other variables.

Branding Ramp

A progressive set of colors based on an application's brand color. In the Oracle look and feel the branding colors are generated by tinting (mixture of a color with white) and shading (mixture of a color with black) the primary brand color (--oj-palette-brand).

Neutral Ramp

A progressive set of neutral colors based on the same hue (--oj-palette-neutral-hue) and saturation (--oj-palette-neutral-saturation), but varied levels of lightness.

Text Color Ramp

A progressive set of text colors based on the same hsl value (--oj-palette-text-base-hsl), but varied levels of opacity.

Accent Colors

A set of accent or special colors for things like data visializations, errors, warnings, etc
