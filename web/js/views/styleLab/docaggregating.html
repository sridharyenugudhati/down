In the aggregating SCSS file you import the files that make up your theme. Note that the order in the aggregating file is important.

  1. server side variables settings file: This settings file is where you can customize server side variables. Due to the way sass default works, the file where you cusomize server side variables must be imported before the JET platform file.
  2. JET platform file
  3. CSS variables settings file: This settings file is where you can customize CSS variables.
